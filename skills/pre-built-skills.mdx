Galagos ships with 207 pre-built skills covering the most common tools and workflows in bioinformatics. Each skill encodes how to run a specific tool or procedure — including parameter defaults, input/output handling, and result interpretation — so Galagos can apply it correctly without any manual configuration from you.

<Tip>
  You don't need to browse or select skills manually. The skill-finder monitors your prompt as you type and automatically applies the most relevant skills for your analysis. Browse the catalogue in your project workspace if you want to see what's available or verify which skills will be used.
</Tip>

| Skill | Description |
|---|---|

| **RoseTTAFold** | Protein structure prediction with three-track neural network architecture |
| **AutoDock Vina** | Molecular docking to predict how small molecules bind to protein targets |

## RNA-seq

Tools for aligning, quantifying, and performing differential expression analysis on bulk RNA-seq data.

| Skill | Description |
|---|---|
| **STAR** | Fast splice-aware RNA-seq read alignment to a reference genome |
| **HISAT2** | Graph-based alignment of sequencing reads with splice-site awareness |
| **featureCounts** | Read counting for genes and genomic features after alignment |
| **DESeq2** | Differential expression analysis using negative binomial distribution modelling |
| **edgeR** | Differential expression analysis for count data with empirical Bayes estimation |
