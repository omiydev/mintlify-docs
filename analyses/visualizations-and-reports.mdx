After an analysis completes, Galagos presents its findings as a structured report that includes publication-ready visualizations, a summary of the methods and parameters used, key statistical results, and relevant literature references. You do not need to request a report separately — it is generated automatically as part of every analysis.

## Visualization types

Galagos generates plots appropriate to the analysis type. The following are the most common output types:

<CardGroup cols={2}>
  <Card title="Volcano plots" icon="chart-line">
    Used in differential expression analysis to display fold change against statistical significance. Points are colored by significance threshold, making it easy to identify up- and down-regulated genes at a glance.
  </Card>
  <Card title="Heatmaps" icon="table-cells">
    Used for clustering, expression profiling, and correlation analysis. Galagos uses hierarchical clustering by default and labels rows and columns automatically from your data.
  </Card>
  <Card title="UMAP plots" icon="circle-nodes">
    Used in single-cell RNA-seq workflows to visualize high-dimensional data in two dimensions. Cell clusters are colored automatically, and Galagos overlays marker gene expression on request.
  </Card>
  <Card title="Bar charts" icon="chart-bar">
    Used for count comparisons, quality control summaries, and categorical data. Axis labels and legends are generated from your data's column headers and sample names.
  </Card>
</CardGroup>

<Tip>
  You can request a specific plot type in your prompt. For example: "Generate a heatmap of the top 50 differentially expressed genes" or "Show a UMAP colored by cluster and by `GAPDH` expression side by side."
</Tip>

## Structured reports

Every completed analysis includes a structured report in the chat. A report contains:

- **Analysis summary**: a plain-English description of the goal, approach, and tools used
- **Parameters**: the exact values passed to each tool (for example, DESeq2 thresholds, GATK filter settings, Scanpy resolution parameters)
- **Key findings**: statistical results, ranked gene lists, variant counts, or cluster assignments, depending on the analysis type
- **Literature references**: citations from PubMed, UniProt, or NCBI that support the methods or findings, linked inline

Reports are structured so you can read them top-to-bottom for a narrative summary, or skip directly to the parameters section to verify reproducibility.

## Publication-ready outputs

All plots generated by Galagos are formatted for direct use in manuscripts and presentations:

- Vector-quality resolution suitable for journal submission
- Clean axis labels, legends, and color schemes
- Consistent font sizes across panels

You do not need to reformat plots in a graphics editor before including them in a paper.

## Downloading results

To download a report or visualization:

<Steps>
  <Step title="Locate the result in the chat">
    Completed reports and plots appear inline in the chat. Each plot has a download icon in its top-right corner.
  </Step>
  <Step title="Download individual plots">
    Click the download icon on any plot to save it as a PNG or SVG file.
  </Step>
  <Step title="Download the full report">
    Click **Download report** at the bottom of the structured report card. This saves the full report — including parameters, findings, and references — as a PDF.
  </Step>
  <Step title="Access files from the Files tab">
    All output files generated during an analysis are also saved to your project's **Files** tab, where you can download them at any time.
  </Step>
</Steps>

<Note>
  Downloaded files remain available in the **Files** tab for as long as your project exists. Analysis outputs count toward your project's storage quota.
</Note>
